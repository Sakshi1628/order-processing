=> Order Processing System

=> Overview
1)This project is a simple and practical Order Processing System built using Java. It is designed to simulate how customer orders are handled in real-world applications. The system allows users to manage customers, products, and orders in an organized way.

2)The main goal of this project is to demonstrate how Java can be used along with a database to perform real-time data operations like storing, updating, and retrieving information.

=> Objectives
1) To understand how real-world order systems work  
2) To practice database connectivity using JDBC  
3) To implement CRUD operations (Create, Read, Update, Delete)  
4) To build a complete mini project using Core Java  

=> Features
1) Add new customers  
2) Add and manage product details  
3) Place orders for customers  
4) Calculate total order amount automatically  
5)Store and retrieve data from database  
6)Track order details efficiently  

=>Technologies Used
1)Core Java– for application logic  
2) JDBC (Java Database Connectivity) – for connecting Java with database  
3)MySQL – for storing data  
4)OOP Concepts – classes, objects, methods  

 => System Working
The system works by connecting a Java application to a MySQL database using JDBC.

1) When a user adds customer or product details → data is stored in database  
2)When an order is placed → system calculates total amount  
3)All data is fetched and displayed using SQL queries  
4)The application handles operations like insert, update, delete, and view  

=> Project Structure 
1) `Customer.java` → handles customer details  
2) `Items.java` → manages product information  
3) `OrderItem.java` → manages order items  
4) `Orders.java` → handles order processing  
5) `Warehouse.java` → manages storage/logistics  
6) `OracleCon.java` → database connection  

=> How to Run the Project
1. Install Java (JDK) and MySQL  
2. Open the project in Eclipse / IntelliJ  
3. Configure database connection in code  
4. Create required tables in MySQL  
5. Run the main class  
6. Start using the system
